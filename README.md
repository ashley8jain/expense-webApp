# Expense webapp

## step by step

* Install flask app

```
virtualenv flask
cd flask
source bin/activate
pip install flask
```

* create hello.py file

* run the flask app

```
python hello.py
```

* when deploy to webserver

```
pip freeze > requirements.txt
```



## Step to run the webapp

* clone the github repo

* install all library

```
pip install -r requirements.txt
```
